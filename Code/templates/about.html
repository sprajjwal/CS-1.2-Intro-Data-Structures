Welcome to this Donald Trump tweet generator. This website was built using FLASK, Jinja, HTML, CSS and Markov's model in python. Markov's Chain uses original texts to generates a special structure that keeps track of every word that came after a certain word. It then uses that to generate new sentences that seem like it was a part of the original text.
